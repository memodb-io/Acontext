Acontext integrates with popular AI frameworks to add session persistence, task extraction, and observability to your agents.

## Quick Start

```bash
curl -fsSL https://install.acontext.io | sh
acontext create my-proj --template-path "python/openai-agent-artifacts"
```

For more examples, see [Acontext-Examples](https://github.com/memodb-io/Acontext-Examples).
