---
title: Learning Space
description: "Group knowledge from sessions and skills into cohesive learning spaces"
---

Learning spaces let you organize agent knowledge by including skills and learning from sessions in a single, referenceable resource.

<Steps>
<Step title="Create a learning space">

<CodeGroup>
```python Python
import os
from acontext import AcontextClient

client = AcontextClient(
    api_key=os.getenv("ACONTEXT_API_KEY"),
)

space = client.learning_spaces.create(
    user="alice@example.com",
    meta={"version": "1.0", "domain": "customer-support"},
)
print(f"Created space: {space.id}")
```

```typescript TypeScript
import { AcontextClient } from '@acontext/acontext';

const client = new AcontextClient({
    apiKey: process.env.ACONTEXT_API_KEY,
});

const space = await client.learningSpaces.create({
    user: "alice@example.com",
    meta: { version: "1.0", domain: "customer-support" },
});
console.log(`Created space: ${space.id}`);
```
</CodeGroup>
</Step>

<Step title="Include skills">

Skills are automatically learned from the sessions associated with the learning space. You can also manually include or exclude them.

<CodeGroup>
```python Python
# Include a skill
client.learning_spaces.include_skill(space.id, skill_id="skill-uuid")

# List all skills in the space
skills = client.learning_spaces.list_skills(space.id)
for skill in skills:
    print(f"  {skill.name}: {skill.description}")

# Remove a skill (idempotent)
client.learning_spaces.exclude_skill(space.id, skill_id="skill-uuid")
```

```typescript TypeScript
// Include a skill
await client.learningSpaces.includeSkill({
    spaceId: space.id,
    skillId: "skill-uuid",
});

// List all skills in the space
const skills = await client.learningSpaces.listSkills(space.id);
skills.forEach(skill => {
    console.log(`  ${skill.name}: ${skill.description}`);
});

// Remove a skill (idempotent)
await client.learningSpaces.excludeSkill({
    spaceId: space.id,
    skillId: "skill-uuid",
});
```
</CodeGroup>
</Step>

<Step title="Learn from sessions">

Associate a session with a learning space. The record is created with `pending` status for future async processing.

<Note>
A session can only be learned by one learning space. Attempting to learn the same session in another space returns a `409 Conflict` error.
</Note>

<CodeGroup>
```python Python
# Learn from a session
result = client.learning_spaces.learn(space.id, session_id="session-uuid")
print(f"Status: {result.status}")  # "pending"

# Check learning status
sessions = client.learning_spaces.list_sessions(space.id)
for s in sessions:
    print(f"  Session {s.session_id}: {s.status}")
```

```typescript TypeScript
// Learn from a session
const result = await client.learningSpaces.learn({
    spaceId: space.id,
    sessionId: "session-uuid",
});
console.log(`Status: ${result.status}`);  // "pending"

// Check learning status
const sessions = await client.learningSpaces.listSessions(space.id);
sessions.forEach(s => {
    console.log(`  Session ${s.session_id}: ${s.status}`);
});
```
</CodeGroup>
</Step>

<Step title="List and filter">

<CodeGroup>
```python Python
# List all spaces
spaces = client.learning_spaces.list(limit=10)

# Filter by user
spaces = client.learning_spaces.list(user="alice@example.com")

# Filter by meta tags
spaces = client.learning_spaces.list(
    filter_by_meta={"domain": "customer-support"},
)

# Paginate
if spaces.has_more:
    next_page = client.learning_spaces.list(cursor=spaces.next_cursor)
```

```typescript TypeScript
// List all spaces
const spaces = await client.learningSpaces.list({ limit: 10 });

// Filter by user
const userSpaces = await client.learningSpaces.list({
    user: "alice@example.com",
});

// Filter by meta tags
const filtered = await client.learningSpaces.list({
    filterByMeta: { domain: "customer-support" },
});

// Paginate
if (spaces.has_more) {
    const nextPage = await client.learningSpaces.list({
        cursor: spaces.next_cursor,
    });
}
```
</CodeGroup>
</Step>

<Step title="Update and delete">

<CodeGroup>
```python Python
# Patch meta (merges with existing)
updated = client.learning_spaces.update(
    space.id,
    meta={"version": "2.0"},
)

# Delete a learning space (skills and sessions are NOT deleted)
client.learning_spaces.delete(space.id)
```

```typescript TypeScript
// Patch meta (merges with existing)
const updated = await client.learningSpaces.update(space.id, {
    meta: { version: "2.0" },
});

// Delete a learning space (skills and sessions are NOT deleted)
await client.learningSpaces.delete(space.id);
```
</CodeGroup>
</Step>
</Steps>
