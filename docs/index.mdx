---
title: "What is Acontext?"
description: "The agent memory stack for production AI Agents"
---
Acontext is the agent memory stack for production AI Agents. It lets you:
- Store and edit session-scoped data (messages, files, artifacts) with built-in context engineering
- Track agent progress, task status, and summaries
- Learn persistent skills from past sessions to improve over time
- Provide tools like sandbox, filesystem, and skills
