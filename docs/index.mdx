---
title: "What is Acontext"
---
Acontext is a simple context storage for building cloud-native AI Agents, which can:
- Stores and edit context data
- Tracks Agent progress and feedback
- Provides tools like sandbox, filesystem, skills
- Learns skills from past sessions to improve over time
